# Bulls vs Cows

### Getting started

Make sure you use the proper node version:

```
$ nvm use
```

Install dependencies:

```
$ npm install
```

Start the client and server and go to `http://localhost:3000/`:

```
$ npm run server
$ npm run client
```

You can also play around with the cli:
```
$ npm run play-cli
$ npm run demo-cli
$ npm run p2p-cli
```

Testing:

```
$ npm run test
$ npm run client-test
```
